Overview

What is it?

Abstraction of sensor data flow for universal use on different visualisation platforms.

Supported Platforms

  • Linux

  • Windows (10, 11)

  • OSX (11.6+)

  • UWP (Universal Windows)

Supported Compilers

  • Clang 15 and above

  • GCC 12 and above

  • vs2019 and above

Supported Languages

  • C/C++ API documentation

Project Page

  • https://github.com/T-PLAY/hub

Contact

  • Mail `TPlay tplay@irit.fr`__

Getting Involved

Everyone is welcome to contribute to hub by submitting bug reports, testing on different platforms, writing examples (see ideas), improving documentation, profiling and optimizing, etc.

Note

When contributing to the hub project you must agree to the MPL 2.0 licensing terms.
