template<class SerializerT = Serializer, class InputOutputT = InputOutput<SerializerT>>
class hub::io::MemoryT< SerializerT, InputOutputT >

The MemoryT class implements input/output features with a any serializer to save data locally in a vector. It's allow to pack/serialize many objects into one vector.
