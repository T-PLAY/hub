OutputStreamServer2 is used to instantiate an OutputSensor.

Parameters
header
streamName[in] is an unique identifier name of stream
port
ipv4[in] is an existing connection to a hub server
Warning
The hub service must be accessible, that means the server hub is running on a machine you know the ip and the oppened port of the service.
Stream you want to start linking (by it name) needs to not exist in the server hub. Streaming with most of one sender is not allowed.
Exceptions
net::Socket::exceptionwhen the server is not found or by loosing connection to the server. Also occur when stream you want to link is already started in the server.
