Detailed Description

template<class SerializerT = Serializer, class InputOutputT = InputOutput<SerializerT>>
class hub::io::ArchiveT< SerializerT, InputOutputT >

The ArchiveT class implements input/output features with a any serializer to save data locally in the unique archive. You can write or read in the archive.

Warning
When you read in the archive, the information is lost. The data is not retained.
